Pocketbyte is an indie video game handheld computer built on the ESP32-S3 microcontroller. It features a built-in display, audio output, USB connectivity, and — most distinctively — a swappable input module system that lets you attach different physical control schemes to suit your game or application. Instead of locking you into a fixed button layout, Pocketbyte is designed to be reconfigured, reprogrammed, and reimagined. The software you run on it is entirely your own.

## Who It's For

Pocketbyte is built for people who want to make things, not just play them. You'll find it a great fit if you are:

- An **indie game developer** looking for a tangible, portable target platform to ship original games on real hardware.
- A **student or self-directed learner** exploring embedded systems, C programming, or game development in a hands-on environment.
- A **maker or hardware enthusiast** who wants to build interactive devices — not just games, but tools, toys, and experiments — using a purpose-built portable platform.

## What You Can Build

Because Pocketbyte runs arbitrary ESP-IDF applications, your software is only limited by the hardware capabilities and your imagination. Common use cases include:

- **Games** — arcade-style games, puzzle games, platformers, or anything that fits the screen and your chosen input module.
- **Interactive apps** — utilities, calculators, music sequencers, or any application that benefits from a small portable form factor.
- **Educational tools** — interactive demos, simulations, or learning aids that you can carry around and share.

## Key Features

<CardGroup cols={2}>
  <Card title="Swappable Inputs" icon="arrows-rotate">
    Attach different physical input modules to change your control scheme — buttons, a d-pad, or custom layouts — without changing your device.
  </Card>

  <Card title="C SDK" icon="c">
    Write your games and apps in C using the official Pocketbyte SDK. The API covers display, input, audio, and system initialization.
  </Card>

  <Card title="ESP-IDF Integration" icon="plug">
    The Pocketbyte SDK is an ESP-IDF managed component. Add it to any IDF project with one command and use the full power of the ESP-IDF ecosystem.
  </Card>

  <Card title="Open Platform" icon="lock-open">
    You control the software entirely. There is no locked bootloader, no mandatory storefront — flash whatever you build directly to the device.
  </Card>
</CardGroup>

<Note>
  Pocketbyte runs on the ESP32-S3, a dual-core Xtensa LX7 microcontroller with Wi-Fi and Bluetooth. All apps are written in C and built using the ESP-IDF framework (v5.x recommended). Familiarity with C and basic embedded development concepts will help you get the most out of the platform.
</Note>
