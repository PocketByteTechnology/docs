In this walkthrough you'll build a complete — if tiny — Pocketbyte game: a colored box that the player can move around the screen using the directional buttons. Along the way you'll touch every layer of the SDK: project setup, initialization, input reading, rendering, and flashing to hardware. By the end you'll have a working game loop you can grow into anything.

## Complete `main.c`

Here is the full source file with all four phases wired together:

```c
#include "pocketbyte/pocketbyte.h"
#include "freertos/FreeRTOS.h"
#include "freertos/task.h"

#define PLAYER_SIZE   8
#define SCREEN_WIDTH  128
#define SCREEN_HEIGHT 80
#define FRAME_MS      16   /* ~60 fps */

static int player_x = (SCREEN_WIDTH  - PLAYER_SIZE) / 2;
static int player_y = (SCREEN_HEIGHT - PLAYER_SIZE) / 2;

/* ── input ─────────────────────────────────────────────────────── */

static void input(void)
{
    pocketbyte_input_update();

    if (pocketbyte_button_held(POCKETBYTE_BTN_UP))    player_y -= 1;
    if (pocketbyte_button_held(POCKETBYTE_BTN_DOWN))  player_y += 1;
    if (pocketbyte_button_held(POCKETBYTE_BTN_LEFT))  player_x -= 1;
    if (pocketbyte_button_held(POCKETBYTE_BTN_RIGHT)) player_x += 1;

    /* clamp to screen bounds */
    if (player_x < 0)                        player_x = 0;
    if (player_y < 0)                        player_y = 0;
    if (player_x > SCREEN_WIDTH  - PLAYER_SIZE) player_x = SCREEN_WIDTH  - PLAYER_SIZE;
    if (player_y > SCREEN_HEIGHT - PLAYER_SIZE) player_y = SCREEN_HEIGHT - PLAYER_SIZE;
}

/* ── update ─────────────────────────────────────────────────────── */

static void update(void)
{
    /* No additional logic yet — movement is handled in input() for
       this simple demo. Add scoring, collisions, and enemies here. */
}

/* ── draw ───────────────────────────────────────────────────────── */

static void draw(void)
{
    pocketbyte_display_clear(POCKETBYTE_COLOR_BLACK);

    pocketbyte_display_draw_rect(
        player_x, player_y,
        PLAYER_SIZE, PLAYER_SIZE,
        POCKETBYTE_COLOR_RED
    );

    pocketbyte_display_draw_text(2, 2, "Move me!", POCKETBYTE_COLOR_WHITE);

    pocketbyte_display_flush();
}

/* ── entry point ────────────────────────────────────────────────── */

void app_main(void)
{
    pocketbyte_init();

    while (1) {
        input();
        update();
        draw();
        vTaskDelay(pdMS_TO_TICKS(FRAME_MS));
    }
}
```

<Tip>
  Resist the urge to add everything at once. Start with the moving box working perfectly, then layer in one feature at a time — a collectible, a score counter, a boundary wall. Each addition is easy to debug when the foundation is solid.
</Tip>
